# Ecommerce-Dry-Fruits-Website

-> Description
This is a full-stack e-commerce web application built using PHP. The platform allows users to browse and purchase Shilajit and dry fruits, while providing an admin panel for managing products and orders.

-> Features
- User Registration & Login System
- Admin Dashboard
- Add / Edit / Delete Products
- Shopping Cart System
- Checkout Functionality
- Category-based Products
- Responsive UI Design

-> Technologies Used
- PHP (Core PHP)
- MySQL
- HTML
- CSS
- JavaScript

-> Project Structure
- `/admin` → Admin panel
- `/config` → Database configuration
- `/assets` → CSS & JS files
- `/images` → Product images

-> Setup Instructions
1. Clone the repository
2. Import database into MySQL
3. Configure database in `/config`
4. Run project on localhost (XAMPP/Laragon)
